Langages

HTML5

HTML est le langage de balisage utilisé pour réaliser la structure de base du site et de toutes ses pages. Comme vous pouvez le remarquer, l’extension .html est présent sur tous les liens (oui, tu peux vérifier).

CSS3

CSS est le langage de style encore nommé feuille de style qui a servi à donner les couleurs et les animations. Toute la mise en forme du site a été faite dans ce langage. Des polices aux couleurs de fonds, tous.

PHP

PHP est le langage coté serveur qui permet de rendre les pages dynamiques et d’interagir avec les utilisateurs. Il permet aux sites d’être plus autonome et de s’adapter en fonction des utilisateurs.

Java Script

Java Script est le langage qui permet de créer des scripts pour dynamiser les sites coté utilisateur/client. De sorte à contrôler le contenu multimédia, d'animer des images, et tout ce à quoi on peut penser.
